# Lumarc Studio

Site institucional e portfólio da **Lumarc Studio**, agência especializada em desenvolvimento web de alta performance e design premium.

## Tecnologias

Este projeto foi construído utilizando as seguintes tecnologias:

- [Vite](https://vitejs.dev/) - Build tool e servidor de desenvolvimento
- [React](https://reactjs.org/) - Biblioteca para interfaces de usuário
- [TypeScript](https://www.typescriptlang.org/) - Superset tipado de JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - Framework de utilitários CSS
- [shadcn/ui](https://ui.shadcn.com/) - Componentes de interface reutilizáveis
- [Framer Motion](https://www.framer.com/motion/) - Biblioteca de animações

## Como rodar o projeto

1.  **Clone o repositório**
    ```bash
    git clone <URL_DO_REPOSITORIO>
    ```

2.  **Instale as dependências**
    ```bash
    npm install
    ```

3.  **Rode o servidor de desenvolvimento**
    ```bash
    npm run dev
    ```

4.  **Acesse no navegador**
    Abra `http://localhost:8080` (ou a porta indicada no terminal).

## Deploy

O projeto pode ser implantado em qualquer plataforma de hospedagem de sites estáticos, como Vercel, Netlify ou Cloudflare Pages, utilizando o comando de build:

```bash
npm run build
```
